Terrain Data


Kwik EFIS provides a Digital Elevation Model (DEM) for the provision of synthetic vision and above ground level altitide (AGL).

The the synthetic vision is built from the USGS 3 arc second Digital Elevation Model data (SRTM30). If enabled, it also allows for some terrain collision detection.
